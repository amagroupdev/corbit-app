# خدمات منصة أوربت (ORBIT)

## 1. خدمة الرسائل النصية SMS
الحل الأسرع والأكثر ضماناً للوصول لجميع الهواتف. مثالي للإشعارات ورموز التحقق.

### المميزات:
- رموز تحقق (OTP) فائقة السرعة
- إشعارات الطلبات والشحنات
- تغطية لجميع الشبكات المحلية والدولية
- تقارير تسليم فورية ودقيقة
- ربط مباشر معتمد من هيئة الاتصالات

### الاستخدامات:
- إرسال رسائل تسويقية لآلاف العملاء
- إرسال رموز التحقق OTP
- إشعارات الطلبات والتحديثات
- تذكيرات المواعيد
- رسائل المناسبات والتهنئة

### للمتاجر والمسوقين:
- لا تحتاج لخبرة تقنية
- اربط متجرك في سلة أو زد بضغطة زر واحدة
- ابدأ إرسال حملاتك التسويقية لآلاف العملاء فوراً
- استيراد جهات الاتصال تلقائياً
- قوالب رسائل جاهزة ومعتمدة
- تقارير دقيقة للأداء (الفتح، النقر)

### للمطورين والتقنيين:
- API متوافق مع معايير REST
- توثيق شامل ومفصل
- مكتبات جاهزة لعدة لغات برمجة
- بيئة اختبار (Sandbox) مجانية

## 2. واتساب للأعمال API (WhatsApp Business API)
تواصل بشكل رسمي وموثق. ارفع نسبة التفاعل مع عملائك عبر منصتهم المفضلة.

### المميزات:
- شات بوت ذكي للرد الآلي
- توثيق الحساب (العلامة الخضراء)
- رسائل تسويقية لا محدودة
- دعم الوسائط المتعددة (صور، فيديو، ملفات)

## 3. O-Time - برنامج الموارد البشرية
منصة متكاملة لإدارة الموارد البشرية تشمل الحضور والرواتب ودورة حياة الموظف بالكامل.

### المميزات:
- إدارة الحضور والإجازات
- أتمتة مسيرات الرواتب
- بوابة الخدمة الذاتية للموظف
- لوحات تحكم وتحليلات فورية

## 4. Gov Gate - بوابة حكومية
بوابة مراسلة مؤسسية آمنة ببنية مخصصة وامتثال كامل وتحكم متقدم.

### المميزات:
- بوابة مراسلة خاصة وآمنة
- صلاحيات دقيقة بحسب الأدوار
- أمان مؤسسي وامتثال تشريعي
- تقارير تشغيلية وسجل تدقيق مفصل

## التكاملات المدعومة
المنصة تتكامل مع أدواتك المفضلة:
- متجر سلة
- متجر زد
- وأنظمة أخرى عبر API
